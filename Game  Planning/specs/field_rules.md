# フィールドルール設計書 v0.1
> 最終更新: 2026-03-28

---

## 1. マッチの流れ

```
00:00  マッチ開始
       └ プレイヤーがフィールドに散らばる
       └ 収束円の収縮が始まる
       └ 脱出ポイントは非表示

03:00  脱出ポイント解放
       └ 5箇所がマップ上に一斉表示
       └ 以降、いつでも脱出可能

～     収束円が段階的に縮小
       └ 円の外はダメージを受け続ける
       └ プレイヤーが中心部へ押し込まれる

20:00  円が完全に閉じる
       └ 残存プレイヤーは強制死亡・所持品ロスト
```

---

## 2. 脱出ポイント

| 項目 | 仕様 |
|---|---|
| 設置数 | 5箇所 |
| 開示タイミング | マッチ開始から3分後、全プレイヤーのマップに一斉表示 |
| 開閉 | 一度開いたら閉じない |
| 脱出方法 | チャネリング式（その場に留まり続ける） |
| チャネリング時間 | 5秒 |
| 中断条件 | チャネリング中に攻撃を受けると中断・最初からやり直し |
| 同時使用 | 複数プレイヤーが同じポイントで同時にチャネリング可能 |
| チーム脱出 | 1人ずつ個別に脱出可能（全員揃う必要なし） |

### 脱出ポイントでの駆け引き
- チャネリング中は**無防備**になるため、他プレイヤーの格好の標的になる
- 脱出しようとする敵を妨害する戦術が生まれる
- 複数人が同じポイントに集まることで**乱戦**が発生しやすい

## 3. 収束円（Circle）

| 項目 | 仕様 |
|---|---|
| 収縮開始 | マッチ開始と同時 |
| 完全閉鎖 | 開始から約20分後 |
| 円の外のペナルティ | 継続ダメージ |
| 完全閉鎖時の処理 | 残存プレイヤーは強制死亡・所持品ロスト |

### 収縮フェーズ（案）
| フェーズ | 時間 | 状態 |
|---|---|---|
| フェーズ1 | 0:00〜5:00 | 広い範囲、ダメージ小 |
| フェーズ2 | 5:00〜10:00 | 中程度に縮小、ダメージ中 |
| フェーズ3 | 10:00〜15:00 | かなり狭くなる、ダメージ大 |
| フェーズ4 | 15:00〜20:00 | 最終収縮、ダメージ最大 |

> ※ フェーズの数・時間・ダメージ量は要バランス調整

---

## 4. マッチの終了条件

以下のいずれかで個人のマッチが終了する：

| 条件 | 結果 |
|---|---|
| 脱出ポイントからの脱出成功 | 所持品を保持してロビーへ帰還 |
| フィールドで死亡 | 所持品をすべてロスト |
| 収束円の完全閉鎖時に残存 | 強制死亡・所持品ロスト |

---

## 5. マッチの緊張感の設計意図

| 時間帯 | プレイヤーの行動 | テンション |
|---|---|---|
| 0〜3分 | 資源集め・敵狩り、脱出ポイントを探す必要なし | 比較的自由 |
| 3〜10分 | 脱出を狙いつつ資源を集め続けるか判断 | 高まり始める |
| 10〜15分 | 円の縮小で行動範囲が制限、脱出ポイントへ向かう判断 | 高緊張 |
| 15〜20分 | 脱出か生存かの最終決断、脱出ポイントでのPvP激化 | 最高潮 |

---

## 6. 収束円の詳細仕様

### 縮小先
**マップ中央固定**。プレイヤーが事前に行動を予測しやすくする。

### フェーズ別ダメージ量（案）

| フェーズ | 時間帯 | 円外ダメージ（/秒） |
|---|---|---|
| フェーズ1 | 0:00〜5:00 | 3 HP/秒 |
| フェーズ2 | 5:00〜10:00 | 8 HP/秒 |
| フェーズ3 | 10:00〜15:00 | 18 HP/秒 |
| フェーズ4 | 15:00〜20:00 | 35 HP/秒 |

> 序盤は「早めに戻ればいい」、終盤は「数秒の滞在が致命傷」になる強さ感を意図

---

## 7. 脱出ポイント詳細仕様

### 配置ルール
- マップ**外周寄り**に5箇所配置（フェーズ1〜2で使いやすい位置）
- 収束円が縮小するにつれて外周の脱出ポイントが圏外になるため、自然に競争が激化する

### チャネリング中の移動
**移動不可**。その場から動けない。

### 脱出成功後の演出
- 脱出したプレイヤーはマップ上から**消える**
- 全プレイヤーのHUDに「〇〇が脱出しました」とシステムメッセージを表示
- 脱出人数をHUDに常時表示（例：「脱出済み: 7 / 30」）
